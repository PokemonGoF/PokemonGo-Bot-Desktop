Getting Started
---------------
###Install
```
git clone --recursive https://github.com/PokemonGoF/PokemonGo-Bot-Desktop
brew install python node
pip install virtualenv
cd PokemonGo-Bot-Desktop
sudo npm install
```

###Run:
```
npm start
```
